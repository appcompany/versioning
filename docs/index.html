The 5 commandments of commiting.

1. The title shall not contain any changelog tags

2. Every line in the message shall contain a tag

3. Only one change per line, and try to keep them short!

4. Everything except names shall be lowercased

5. Follow rules 1, 2, 3 and 4 religously
